Global Density Dot Plot (2022)

Figure 1 shows a world map that shows the
airports that range between ~20 flights to ~130
flights. Each proportional symbol is accompanied by information
relating to the Airport's: Airport's Name, Frequency and Location.

Most Active Months

Figure 2 shows a stacked bar chart to represent the
top coutries around the world with the most active flights
for each period of each month

Monthly Average Ages (Per Continent)

Figure 3 shows an interactive grouped/multiple bar chart.
It allows you to see the average age per each
flight for each month in 2022.
